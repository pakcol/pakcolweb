Stivent Nathaniel Riwulaky

Freelance Beginner

Saya lahir di Surabaya, Jawa Timur, Indonesia. Saya tinggal di Perbatasan Gresik - Surabaya lebih tepatnya di Driyorejo.

Saya mengikuti Sekolah Dasar (SD) dan Sekolah Menengah Pertama (SMP) di SD - SMP Katolik Santo Yosef Tarakanita Surabaya selama 6 tahun SD dan 3 tahun SMP. Saya di sini belajar cukup banyak termasuk saya tertarik dengan pelajaran mengenai Komputer dimana saya diajarkan basic menggunakan software CorelDraw, Adobe Flash, Photoshop dan membuat Blogger.

Kemudian saya melanjutkan Sekolah Menengah Kejuruan di SMK Kristen Petra Surabaya. saya menempuh pendidikan di sini selama 3 tahun. Saya masuk ke dalam jurusan Teknik Komputer dan Jaringan. Saya di sini belajar bahasa pemrograman C++ dan juga belajar mengenai infrastruktur jaringan termasuk belajar MikroTik. Pada tahap sekolah SMK ini juga saya magang di salah satu instansi pendidikan yaitu Akademi Kebidanan Griya Husada Surabaya, yang dimana sekarang sudah berubah menjadi Sekolah Tinggi Kesehatan Griya Husada Surabaya.

Setelah saya menyelesaikan sekolah saya di SMK, saya melanjutan studi di Universitas Surabaya. Di sini saya masuk ke dalam Program Studi Teknik Informatika jurusan Game Development. Dimana sekarang saya memasuki semester 4 yang sudah menempuh 1 mata kuliah jurusan saya yaitu Game Concept Design, dimana saya mempunyai project membuat game menggunakan Game Maker.

Di masa kuliah ini saya memanfaatkan waktu senggang dimana saya mengikuti beberapa organisasi dan kepanitiaan yang bisa mengasah skill saya dari segi kepemimpinan hingga skill editing saya.
